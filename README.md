# KyMM - Kyria's Mod Manager

A mod manager for **Black Desert Online**. Inject, restore, and manage your mods without ever touching the game's original files.

---

## Features

- All regions supported
- One-click inject and restore
- Mod priority and conflict detection
- Category system with [Tag] grouping
- Game file explorer with wildcard search
- CharacterAO & PartCutDesc auto-generators
- Profile support

---

## Download

Head to the [Releases](../../releases) page and download the latest `.exe`.

No installation needed - just put it into whatever folder you want and run it.

---

## Getting Started

1. Launch KyMM
2. Go to **Settings** and point **PAZ Folder** to your Black Desert `Paz` directory
3. Drop your mod folders into `Mods/`
4. Press **INJECT**

For a full guide, click **How To Use** inside the app.
